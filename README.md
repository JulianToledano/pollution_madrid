## Primeros pasos
Para poder ejecutar el notebook se debe tener instalado `jupyter` y los paquetes `numpy` y `pandas`. Para ejecutarlo sigue los siguientes pasos:

`virtualenv my_env`

`source my_env/bin/activate`

`pip install -r requirements.txt`

`jupyter notebook`
